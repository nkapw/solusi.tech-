# Landing Page Nyiroro — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use `superpowers:subagent-driven-development` (recommended) or `superpowers:executing-plans` to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a complete, production-ready Astro.js landing page for Nyiroro (software development agency) with full SEO, Classic Classless design, and zero client-side JavaScript.

**Architecture:** Single `index.astro` page composing 11 focused `.astro` components. `BaseLayout.astro` handles all SEO meta tags and JSON-LD schemas. Tailwind CSS is used only for responsive layout (grid/flex); all visual styling lives in scoped `<style>` blocks or `global.css`.

**Tech Stack:** Astro.js (latest), Tailwind CSS via `@astrojs/tailwind`, `@astrojs/sitemap`, `@astrojs/partytown`, TypeScript (strict)

---

import { defineConfig } from 'astro/config';
import tailwind from '@astrojs/tailwind';
import sitemap from '@astrojs/sitemap';
import partytown from '@astrojs/partytown';

export default defineConfig({

  integrations: [
    tailwind({ applyBaseStyles: false }),
